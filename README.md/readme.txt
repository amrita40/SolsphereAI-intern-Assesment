Overview
This is the Admin Dashboard for the Cross-Platform System Utility project. It provides a web interface to view and manage system health data collected from multiple machines.

The dashboard displays:

A list of all reporting machines

Latest system health status for each machine (disk encryption, OS updates, antivirus, inactivity settings)

Flags configuration issues clearly (e.g., unencrypted disk, outdated OS)

Shows the last check-in time

Allows filtering and sorting by OS, issue status, and other parameters

Features
Responsive React web app using React.js

Tailwind CSS for styling

Fetches machine data from backend API (or uses mock data if backend not available)

Dynamic filtering and sorting of machine list

Visual indicators for system health issues
